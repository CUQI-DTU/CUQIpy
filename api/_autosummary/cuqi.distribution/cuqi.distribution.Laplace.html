Laplace#

class cuqi.distribution.Laplace(location, scale, **kwargs)#

Laplace distribution.

Defines a Laplace distribution given a location and a scale. The Laplace distribution is defined as

\[p(x) = \frac{1}{2b} \exp\left(-\frac{|x-\mu|}{b}\right),\]

where \(\mu\) is the location (mean) and \(b\) is the scale (decay) parameter.

The rate parameter is defined as \(\lambda = \frac{1}{b}\).

The variables of this Laplace distribution are independent identically distributed (i.i.d.).

Parameters:

  • location (scalar or ndarray) – The location parameter of the distribution.

  • scale (scalar) – The scale parameter of the distribution.
